Favmeal Recipe

2022
Java Script

An small app to explore meals. You can find what you like to eat and add it to favorite.
https://github.com/yardnight/Favmeal-Recipe

Are you intrested to find out how it works? Press the button below to follow the pege link.

There is used an public API from https://www.themealdb.com/api.php.
The API alows to serch meal from a meal list, to get meal by its ID and to use an random meal. If you click either on a random mealor or a favorite one from the favorite meal list then apear informtion popup about the meal. There are you can find all meal ingradients and recipe how to cook it.

Also was added a bit an adaptive design for mobile devices, but only for the most common ones.
