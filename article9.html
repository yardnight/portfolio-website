The Tic-tac-toe

2022
React App

An App of a popular game The tic-tac-toe. The game provides ability to write a history of the current game and then to return to previous game steps. The game history could be called in a popup. The app is based on the first task of official react tutorial with the all propose improvements in it.
https://github.com/yardnight/the-club

I've started to learn React with the official tutorial. There was an app of the tic tac toe game. In the end of the first task there were propositions of App improvements. When I finished with all of them then i added a slight UI and UX to nice look. Also there were implemented both a popup for showing the game history and device adoptation for different devices
