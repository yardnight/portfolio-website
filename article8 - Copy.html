React was a something unknown for me. So I start to search simple examples how to use it. Then I found a video on Youtube how to create an App for a sport club. That was a webpage written on React. It was not a simple one but useful to show me how react and its components work. When I create the page I wonder if i can put it in github as a usual project as did before therefore I published github page after had run common build.

In the end it was just a blank page with nothing in. Not so easy I thought. So i keep search the solution and I find the theory github-pagesthis and a video: TelmoSampaio how to implement deploy. As it turned out there is a plugin which ables to deploy app to github - gh-pages. But it didnt help me 'cause I made the deploy a several times before and my cach was overflowed. Then I found in stackOverflow how to clean it. Now solution was find and I can deploy apps to github.
